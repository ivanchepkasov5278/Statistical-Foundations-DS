## 🛠️ Quick Start (General)
To explore the various interactive tools (like the Bond Pricing engine):
1) Clone the repository: `https://github.com/ivanchepkasov5278/Statistical-Foundations-DS.git`.
2) Navigate into a specific module folder.
3) Use the provided ./run.sh or ./run.bat scripts to launch the local environment.

## 🧠 Why This Repository?
The goal of this collection is to demonstrate **"White-Box" implementation**. Instead of just calling libraries, these projects focus on:
- **Manual Implementation**: Building the logic from scratch where possible.
- **Interactivity**: Using Streamlit to turn static formulas into dynamic, visual simulators.
- **Theoretical Rigor**: Documenting the "Why" behind the "How" in each sub-directory's README.
